# Exclusive Veículos - Catálogo

Catálogo digital premium para a Exclusive Veículos, desenvolvido com React, Vite, TailwindCSS e Supabase.

## Funcionalidades
- Catálogo de veículos com filtros avançados
- Área administrativa segura
- Integração com Supabase para dados e armazenamento de imagens
- Design responsivo e moderno

## Tecnologias
- React + Vite
- TailwindCSS
- Supabase (PostgreSQL + Auth + Storage)
